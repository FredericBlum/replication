Key to R scripts for data preparation and analysis

01_formatting.R
  Data preparation: converts the dataset in "full_dataset.csv" into the long format for modeling

02_models.R
  Modeling: prepares an aggregated dataset, builds the model, saves fitted and observed values converted to log-odds, saves cardinal sounds, saves all plots

03_plots.R
  Compiles a pooled table of fitted values from all models and plots them
